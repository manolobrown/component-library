# Typography

Our typography system is designed for clarity and readability.

## Font Families

### Sans-Serif (Inter)

The primary font family used for all UI text.

## Usage Guidelines

### Headings
- Use **semibold (600)** for page titles and section headers
- Use **medium (500)** for card titles and sub-headings

### Body Text
- Use **regular (400)** for body text and descriptions
- Use **text-sm (14px)** for most UI text
- Use **text-base (16px)** for longer form content

### Labels & Captions
- Use **medium (500)** for form labels
- Use **text-xs (12px)** for captions and helper text
