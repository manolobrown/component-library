## Usage Guidelines

- **neutral-900** - Primary text color
- **neutral-600** - Secondary text color
- **neutral-400** - Placeholder and disabled text
- **neutral-200** - Borders and dividers
- **neutral-50** - Page backgrounds
- **primary-600** - Primary buttons and links
- **primary-500** - Hover states
- **error-600** - Error states and destructive actions
